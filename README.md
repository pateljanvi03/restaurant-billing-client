# Restaurant Billing Client

This is frontend learning project made for Restaurant Manager to manage Billing & Tables. Built using Vue.js, Router & Tailwindcss.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
